# Puppeteer Server

> Koa + TypeScript + Rollup + Puppeteer

## Project structure


```
.
├── src
│   ├── config.ts
│   ├── controllers
│   ├── main.ts
│   ├── routes.ts
│   └── services
├── ecosystem.config.js
├── nodemon.json
├── package.json
├── rollup.config.ts
└── tsconfig.json
```

## Install

```bash
yarn
```

## Run


```bash
yarn dev
```
